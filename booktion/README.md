# Booktion
Användarmanual
Vid behov, ladda ner SpringBoot, Java, Maven. API-nycklarna finns tillagda i application.properties i zip:en men inte i repositoryt. 
Klona projektet: git clone https://github.com/link 
Gå till src/main/java/com/api/controller/BooktionApplication.java 
Bygg och starta servern: mvn spring-boot:run eller klicka på Run i IntelliJ.
